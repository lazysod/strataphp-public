# ⚠️ CMS Integration

The Media module is required by the CMS module for image and media management features.

**If disabled:**
- CMS media features (uploads, modal, library) will not work
- A warning will be shown in the CMS admin panel

**Integration points:**
- Media manager modal in CMS page editor
- Image uploads and selection
- Media library UI
# Media Module

Handles image and file uploads, management, and media library UI for StrataPHP.

## Features
- Image and PDF upload
- Thumbnail generation
- Media library UI
- Integration with CMS module

## Usage
Enable the module in `modules.php` and ensure it is required by the CMS module.
