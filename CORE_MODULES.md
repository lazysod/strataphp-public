# Core Modules Overview

StrataPHP ships with a set of core modules that provide essential functionality for most web applications. Here’s a summary of each module and its purpose:

## Admin
- Central dashboard for site management
- User and module administration
- Access to CMS and analytics

## User
- User registration, login, and profile management
- Device-based session tracking
- Admin account creation

## StrataCms
- Content Management System (CMS)
- Page creation, editing, publishing
- SEO metadata and template selection

## Links
- Manage site-wide links and navigation
- Link categories and sorting

## Media
- File and image uploads
- Media library management

## Contact
- Contact forms and email handling
- Form validation and spam protection

## GoogleAnalytics
- Integration with Google Analytics for site tracking

## OAuthClients
- OAuth client management for third-party authentication

## HelloWorld
- Example module for learning and testing

## API
- RESTful API endpoints for CMS and other modules

## Home
- Default homepage module

## CookieBanner
- Cookie consent management for GDPR compliance

---

Each module can be enabled, configured, or extended via the admin panel or CLI tools. For custom modules, see the Module Development guide.
